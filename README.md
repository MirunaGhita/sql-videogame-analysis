The goal of this analysis is to explore and visualize video game trends, including:

1. **Top Games, Genres, and Platforms per Region**  
   - Identify the best-selling games, most popular platforms, and top genres in North America, Europe, Japan, and globally.

2. **Decade-Wise Platform Leaders**

3. **Genre Mood Tracker**

## **Techniques & Features**

- SQL queries with aggregations (`SUM`, `ROUND`, `GROUP BY`, `TOP`) to rank sales data.  
- Common Table Expressions (CTEs) and window functions (`RANK() OVER`) for analyzing platform dominance by decade.  
- Conditional logic to assign moods to genres and analyze trends regionally and globally.  
- Provides actionable insights for game developers, marketers, and analysts.
